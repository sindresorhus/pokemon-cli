# pokemon-cli

> Get [Pokémon](https://en.wikipedia.org/wiki/Pok%C3%A9mon) names


## Install

```
$ npm install --global pokemon-cli
```


## Usage

```
$ pokemon --help

  Examples
    $ pokemon
    Snorlax
    $ pokemon --all
    Bulbasaur
    Ivysaur
    …
    $ pokemon --language=ja
    マクノシタ

  Options
    --all       Get all the names instead of a random name
    --language  Set the language for the pokemon  [Default: en]
```
